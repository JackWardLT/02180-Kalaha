Currently set up as fully manual 2 player game (AI mode not available).
Player0 (Future You) and Player1 (Future AI)
(Naming will be changed, but current naming matches indexing in some arrays - seemed easier to read.)

-------------------------------------------
TO RUN:
1)navigate to folder Code
2)execute -- python BaseCode.py

-------------------------------------------
3)This will prompt you to enter number of Cups/ Bins/ Fields per player and number of stones/pits/seeds per Cup.

4)In next prompt you can choose which player will make the first move (By default your move is first, if the prompt is disabled)

5)Game begins - just follow the prompts to see which Player goes next. 
  Enter the index of the field you want to choose as your move as an integer (list shows available options for each player)

6) See if you can break it! 

NOTE! There are currently no restrictions for selecting a move outside of the allowed index range - I will add it today

--------------------------------------------
IF YOU NEED TO BRUTE FORCE STOP THE GAME:
		Press Ctrl+C
